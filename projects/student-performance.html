Predictive Analytics & Machine Learning

Student Performance Prediction

A predictive analytics platform designed to analyze historical educational data, identifying students at risk of academic failure using ensemble machine learning models.

Timeline

Fall 2024

Category

Predictive Analytics

Codebase

Academic Repository (Private)

Overview & Problem

In modern education, early intervention is key to supporting students struggling with academic performance. However, schools and teachers often lack the tools to identify struggling students early enough in the semester to apply meaningful corrections.

This project develops an end-to-end predictive machine learning model to analyze historical records (grades, course enrollment, test scores, attendance patterns, and study habits). The goal is to accurately forecast final student marks and flag individuals who may fail, enabling instructors to intervene proactively.

Data Pipeline & Modeling

The project pipeline covers data preprocessing, scaling, balancing, and model optimization:

Data Preprocessing: Handles outliers, interpolates missing grade variables, and scales numerical values. Encodes categorical inputs (e.g. course departments) using standard one-hot encodings.
SMOTE Rebalancing: Academic datasets are highly imbalanced, with far fewer student failures than passing marks. To prevent models from learning a bias towards passing grades, we applied Synthetic Minority Over-sampling Technique (SMOTE) during the training phase.
Ensemble Classifiers: Trains multiple classifiers, including Random Forests and gradient-boosted trees (XGBoost). The final classifier uses soft-voting ensembling, reaching a peak accuracy of 91.8% and a high recall rate of 89.2% to ensure at-risk students are rarely missed.

Insights & Interpretability

Predictive accuracy is only part of the solution; educators need to know *why* a student was flagged to take appropriate action. To provide this transparency, we integrated SHAP (SHapley Additive exPlanations) values to explain individual predictions.

The analysis showed that class attendance, mid-term grade trajectories, and weekly study hours were the most significant predictive features. The dashboard displays these insights in a clean interface, grouping students into clear risk tiers and highlighting primary risk factors for each flag.

Tech Stack Detail

Python

Primary programming language for dataset cleaning, pipeline construction, and training scripts.

Scikit-Learn

Core machine learning library used for data preprocessing, pipeline structures, Random Forest training, and metrics reporting.

XGBoost

Gradient-boosted decision tree algorithm used to maximize predictive precision and robustness on complex tabular parameters.

SHAP

Model interpretability framework used to measure feature contributions and generate user-friendly risk summaries for teachers.
